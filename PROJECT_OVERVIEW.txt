PROJECT OVERVIEW - ColorArt

==============================
1. IMPLEMENTATION DESCRIPTION
==============================
ColorArt is an AI-powered interactive coloring application built with React Native. It transforms user-uploaded photos or patterns into "Color by Number" digital canvases. 

Key Flow:
- Image Acquisition: Users pick images via `react-native-image-picker`.
- Edge Detection: A custom native module treats the image to extract high-contrast outlines.
- Gameplay: An interactive Skia-based canvas allows users to tap regions and fill them with colors, maintaining the "Line Art" integrity.
- Persistence: Works are saved locally using MMKV and can be viewed in a Gallery.

==============================
2. LIBRARIES & TECHNOLOGIES
==============================
- CORE FRAMEWORK: React Native (v0.83.1)
- GRAPHICS: @shopify/react-native-skia (High-performance 2D drawing)
- STATE MANAGEMENT: Zustand (Lightweight, hook-based state)
- DIRECT STORAGE: react-native-mmkv (Ultra-fast key-value storage)
- ANIMATIONS: react-native-reanimated (60fps UI interactions)
- NATIVE BRIDGE: react-native-nitro-modules (Next-gen JS-to-Native communication)
- BACKEND: Vercel (Python serverless functions for image processing)
- NETWORKING: Axios

==============================
3. ALGORITHMS & TECHNIQUES
==============================
- EDGE DETECTION: 
    * Sobel Operator (implemented in ImageProcessorModule.java/m). 
    * Techniques: Grayscale conversion, Convolution matrices (3x3 kernels), Gradient Magnitude calculation.
- REGION SEGMETATION:
    * Grid-based region mapping to identify "Color by Number" zones.
- RENDERING:
    * Layered rendering using Skia Shaders and Paths to ensure colors don't bleed over black outlines.
